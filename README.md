## Installation

### Framework herunterladen
1. im Terminal mit dem Befehl **cd** in den **Documents** Ordner navigieren
2. folgenden Befehl kopieren, einfügen und Enter drücken

```bash
curl -s "https://laravel.build/example-app" | bash
```
